üöÄ Getting Started

What is the 100 Days of Web Development Challenge? ‚ñº
The 100 Days of Web Development Challenge is a structured learning program designed to help you master web development through daily practice. Each day focuses on building projects, learning new concepts, and improving your coding skills.
How do I start the challenge? ‚ñº
Simply create an account, explore the dashboard, and start with Day 1. Each project includes instructions, code examples, and learning objectives to guide you through the process.
Do I need prior coding experience? ‚ñº
No! The challenge is designed for all skill levels. We start with basic HTML, CSS, and JavaScript concepts and gradually progress to more advanced topics.

üìä Progress Tracking

How does the progress tracker work? ‚ñº
The progress tracker uses a GitHub-style heatmap to visualize your daily activity. Green squares indicate completed days, with darker colors showing higher activity levels. Click on any day to mark it as complete.
What counts as completing a day? ‚ñº
A day is considered complete when you finish the assigned project, understand the concepts, and can demonstrate the functionality. You can mark days as complete using the "Mark Today Complete" button.
How are streaks calculated? ‚ñº
Streaks count consecutive days of activity. Your current streak shows how many days in a row you've been active, while your best streak shows your longest achievement.

üîß Technical Issues

The website is not loading properly ‚ñº
Try refreshing the page, clearing your browser cache, or using a different browser. Make sure JavaScript is enabled in your browser settings.
My progress is not saving ‚ñº
Progress is saved locally in your browser. Make sure you're not in incognito/private mode and that your browser allows local storage. You can export your progress from Settings as a backup.
Projects are not working on mobile ‚ñº
Most projects are optimized for mobile, but some advanced features may work better on desktop. Try using a modern mobile browser like Chrome or Safari for the best experience.

How do I change my profile information? ‚ñº
Go to Settings from the user menu, update your display name, email, and bio, then click "Save Changes" to update your profile.
Can I export my progress data? ‚ñº
Yes! In Settings > Data Management, you can export all your progress data as a JSON file. This includes your completed days, settings, and statistics.
How do I reset my progress? ‚ñº
In Settings > Data Management, click "Reset All Progress". You'll need to type "RESET" to confirm. This action cannot be undone, so make sure to export your data first if needed.
