# Day 178: Planet Terraformer Lite

## Description
A mini strategy simulation game where you modify terrain to sustain life on a planet. Features real-time terrain deformation using mouse tools and keyboard shortcuts.

## Features
- **Real-time Terrain Deformation**: Click and drag on the canvas to raise or lower terrain.
- **Life Meters**: Monitor Atmosphere, Water, and Heat levels that change based on terrain modifications.
- **Tools**: Raise Terrain, Lower Terrain buttons.
- **Disaster Mode**: Simulate disasters that randomly alter terrain and meters.
- **Controls**: Mouse for deformation, keyboard shortcuts (R for raise, L for lower, D for disaster).

## How to Play
1. Use the mouse to deform the terrain by clicking and dragging.
2. Switch tools using buttons or keyboard (R/L).
3. Try to balance the meters to sustain life.
4. Activate disaster mode to add challenge.

## Technologies Used
- HTML5 Canvas for terrain rendering
- JavaScript for game logic
- CSS for styling

## Files
- `index.html`: Main HTML structure
- `style.css`: Styling
- `script.js`: Game logic and interactions
