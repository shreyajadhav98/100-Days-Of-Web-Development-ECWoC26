# Day 163: Asteroid Forge

A micro survival crafting game in zero-G space.

## Features
- 3D asteroid field navigation
- Land on drifting asteroids
- Mine resources with oxygen timer
- Craft tools (Pickaxe, Drill)
- Keyboard movement (WASD, Space, Shift) + mouse look
- Press E to land/take off
- Press M to mine when landed

## Controls
- **WASD**: Move forward/backward/left/right
- **Space**: Move up
- **Shift**: Move down
- **Mouse**: Hold and drag to look around
- **E**: Land on asteroid / Take off
- **M**: Mine resources (when landed)

## Game Mechanics
- Oxygen depletes over time
- Land on asteroids to mine resources
- Use resources to craft better tools
- Survive as long as possible!

## Technologies
- Three.js for 3D graphics
- HTML5 Canvas
- JavaScript ES6

Open `index.html` in a modern browser to play.
