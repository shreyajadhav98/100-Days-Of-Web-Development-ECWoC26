# Day 164: Titan Climb

Climb a moving stone titan while avoiding attacks in this vertical traversal boss encounter.

## Features
- 3D stone titan with handholds to climb
- Stamina meter that depletes while climbing
- Weak-point strikes to damage the titan
- Titan moves and rotates slowly
- Mouse click to climb handholds
- Keyboard movement and jumping

## Controls
- **WASD**: Move camera around
- **Mouse**: Hold and drag to look around
- **Mouse Click**: Click on handholds to climb
- **Space**: Jump off current handhold

## Game Mechanics
- Climb handholds to ascend the titan
- Red handholds are weak points - clicking them damages the titan
- Stamina decreases while on handholds, regenerates when off
- Defeat the titan by reducing its health to 0
- Titan moves, making climbing challenging

## Technologies
- Three.js for 3D graphics
- HTML5 Canvas
- JavaScript ES6

Open `index.html` in a modern browser to play.
